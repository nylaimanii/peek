# peek

peek inside a neural network.

an interactive learning experience for mechanistic interpretability. train a tiny neural net in your browser, then peel it apart with the same techniques used at the frontier of AI research — sparse autoencoders, attribution graphs, activation steering. no setup, no PhD required.

## status

in active development. v0.1 (project scaffold) — done.

## stack

- next.js 16 (app router, turbopack default)
- typescript 5
- tailwind v4
- tensorflow.js (in-browser training, added in step 3)
- react flow (network graph viz, added later)
- d3 (scatterplots + loss curves, added later)
- framer motion (animations, added later)
- zustand (global state, added later)

## dev

```bash
npm run dev
```

opens at http://localhost:3000
